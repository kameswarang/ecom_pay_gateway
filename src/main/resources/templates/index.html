Working Features

  • Homepage is /home
  • REST Endpoints are at /api
    • List all books /api/books
      • Display paginated and sorted results
      • ?page=0,1,2 can be used to get subsequent pages
      • ?size can be used to customise number of results per page; defaults to 20
      • ?sort=field1,(asc|desc)&field2,(asc|desc) can be used to sort based on fields/li>
      • Example 1: /api/books?page=0&sort=price,desc
      • Example 2: /api/books?page=3&sort=title,asc
    • Search based on text /api/books?text
      • Performs indexed search on fields title and authors
      • Example 1: /api/books?text=mockingbird
      • Example 2: /api/books?text=rowling
      • Phrased search is not implemented; "Harry Potter" will search for 'Harry' & 'Potter', not the whole phrase
    • Get images for books /api/bookImages
  • Payment system integrated

Features that were not implemented due to time constraint

  • UI to interact with REST endpoints for searching, pagination and sorting
  • User Authentication
